# Related Projects

Color Highlighter is part of the **Atom Material** family of themes and plugins
by [Elior "Mallowigi" Boukhobza](https://github.com/mallowigi) and the
[AtomMaterialUI](https://github.com/AtomMaterialUI) organization. If you enjoy
this plugin, you might like these too.

## JetBrains Plugins

- **[Material Theme UI](https://plugins.jetbrains.com/plugin/8006-material-theme-ui)**
  — a complete redesign of the JetBrains IDE UI with a large collection of dark
  and light Material Design themes and color schemes.
  ([Lite version](https://github.com/AtomMaterialUI/material-theme-ui-lite) with
  just the themes and color schemes is also available.)
- **[Atom Material Icons](https://github.com/AtomMaterialUI/a-file-icon-idea)**
  — modern, visually distinct Material Design file, folder, and UI icons for
  better visual grepping.
- **[Image Icon Viewer](https://github.com/AtomMaterialUI/Image-Icon-Viewer)**
  — preview images and icons directly from the Project View.

## Other Platforms

The Atom Material tools reach beyond JetBrains IDEs:

- **[Atom Material Icons for VS Code](https://github.com/AtomMaterialUI/a-file-icon-vscode)**
  — the icon pack for Visual Studio Code.
- **[Atom Material Icons for the Web](https://github.com/AtomMaterialUI/a-file-icon-web)**
  — a browser extension bringing the icons to GitHub, GitLab, Gitee and
  BitBucket.
- **[Color Highlighter for the Web](https://github.com/AtomMaterialUI/color-highlighter-web)**
  — a web port of this very plugin.

## Under the Hood

- **[Icon Generator](https://github.com/AtomMaterialUI/iconGenerator)** — the
  shared icon collection powering the Atom Material Icons plugins across
  JetBrains, Chrome and Firefox.
